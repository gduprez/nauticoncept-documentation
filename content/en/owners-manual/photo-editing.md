# Photo Editing

![Photo Editing](../../media/manuel-proprietaires/nauticoncept_photoedit2.png)

In full screen, swipe to navigate between photos and zoom in on the image. A menu is available by clicking on the photo area:

- **Display in HD**
- **Set as main photo**
- **Info**

Editing displays the date taken and allows you to edit tags. Default and frequent tags are suggested. You can add custom tags and delete tags via the trash can icon.
