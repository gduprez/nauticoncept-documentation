# Home and Menu

![Owner Home Page](../../media/manuel-proprietaires/nauticoncept_accueil_menu.png)

The home page displays the list of boats with photos, makes, models, names, and registrations. A search and filtering tool facilitates navigation for multiple vessels.

## Burger Menu

By clicking on the 3-line icon (burger menu), you access the following features:

- **My Dealer**
- **My Fleet**
- **Internal Messaging**
- **Alerts**: for owners of surveillance modules
- **My Schedule**
- **My Account**
- **About**
- **Log Out**
