# Location

![Fleet Location](../../media/manuel-concessionnaires/nauticonept_flotte_local.png)

The map view shows all the boats for which you have entered the usual location on the map. If you have the NauticSafe monitoring module, the real-time location of the boat will be displayed.

## Visual Indicators

Boats for which work is in progress appear with an orange flag. By clicking on a flag on the map, information about the boat appears and you can access the boat's details.

> **Note:** When you edit the usual position of the boat, it is possible to move the flag on the map to correct GPS system inaccuracies.
