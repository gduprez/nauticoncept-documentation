# Localisation

![Localisation du bateau](../../media/manuel-proprietaires/nauticoncept_localisation.png)

Les informations de localisation incluent la position habituelle accessible dans la fiche technique. Avec un module de surveillance, la localisation temps réel est disponible.

> **Note :** Il est possible de corriger les imprécisions GPS en déplaçant le drapeau sur la carte.
