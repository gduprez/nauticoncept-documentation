# Mon compte

![Mon compte](../../media/manuel-proprietaires/nauticoncept_moncompte.png)

La page Mon Compte permet l'affichage et la modification de vos coordonnées personnelles : noms, prénoms, adresse, téléphone.

Vous pouvez également modifier votre code PIN et votre mot de passe.
