# Messagerie

![Messagerie](../../media/manuel-proprietaires/nauticoncept_messagerie.png)

La messagerie est un outil embarqué à l'application NAUTICONCEPT pour garder trace de vos discussions avec votre concessionnaire.

Il s'agit d'un fil de discussion pour poser des questions, prévenir de visites, partager des informations sur l'entretien.

## Accès

Accessible via le menu Burger ou la page d'accueil.

## Indicateurs visuels

- Messages non lus indiqués en rouge
- Enveloppes noires pour les messages non lus
- Enveloppes blanches pour les messages lus

## Créer un message

Cliquez sur le bouton **+** pour créer un nouveau message.

## Pièces jointes

Vous pouvez joindre des photos via le trombone pour appuyer vos explications.
