# Mon bateau

![Fiche bateau](../../media/manuel-proprietaires/nauticoncept_fichebateau.png)

Si vous cliquez sur un bateau de la liste, la fiche technique détaillée s'affiche. Dans cette fiche, vous pouvez retrouver toutes les informations concernant :

- Son identité
- Sa localisation habituelle
- Ses dimensions et poids
- La description de ses moteurs
- Les équipements à bord

La section « Documents » permet de stocker les factures et fiches techniques.

## Partage avec votre concessionnaire

Les informations sont partagées avec le concessionnaire pour faciliter l'entretien de votre bateau.

## Démarrer une intervention

Un bouton « Démarrer une intervention » permet d'enregistrer les entretiens personnalisés.
