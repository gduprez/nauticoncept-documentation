# Recherche de bateaux

![Filtres de flotte](../../media/manuel-concessionnaires/nauticoncept_filtreflotte-1.png)

Deux méthodes de filtrage sont disponibles pour retrouver facilement un bateau dans votre flotte.

## Filtres par type de bateau

Dans la fiche technique du bateau, un champ filtre vous permet de classifier les bateaux selon leur grande typologie : voilier, hors bord, semi-rigide, inboard, habitable...

Ces filtres sont disponibles depuis le petit icône entonnoir situé en haut à droite de l'écran. La liste des bateaux est automatiquement filtrée dans cet écran. Pour cacher les filtres, appuyez à nouveau sur l'icône entonnoir.

## Zone de recherche

Vous pouvez également rechercher un bateau par les champs suivants :

- Marque
- Modèle
- Immatriculation
- Nom du bateau
- Nom du client
