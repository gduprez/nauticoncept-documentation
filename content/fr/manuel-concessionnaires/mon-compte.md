# Mon compte

![Mon compte](../../media/manuel-concessionnaires/nauticoncept_moncompte.png)

La page Mon Compte affiche vos coordonnées personnelles et permet de les modifier.

Pour modifier vos noms, prénoms, adresse et téléphone, cliquez sur « Modifier mes informations personnelles ».

Vous pouvez également modifier votre code PIN ainsi que votre mot de passe.
