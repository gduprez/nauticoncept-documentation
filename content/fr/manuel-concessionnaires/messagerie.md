# Messagerie

![Messagerie](../../media/manuel-concessionnaires/nauticoncept_messagerie.png)

La messagerie est un outil embarqué à l'application NAUTICONCEPT pour garder trace de vos discussions avec vos clients. Il s'agit d'un fil de discussion dans lequel vous pouvez partager des informations sur vos maintenances, demander une validation pour une intervention, prévenir votre client sur un souci quelconque, etc.

## Accès

La messagerie est accessible depuis le menu Burger, ou directement sur la page d'accueil.

## Indicateurs visuels

Le nombre de messages que vous n'avez pas encore lus est indiqué en rouge sur l'onglet messagerie. Les messages non lus sont symbolisés par une enveloppe noire, et blanche pour les messages lus.

## Créer un message

Pour écrire un nouveau message à votre client, cliquez sur le bouton **+** en haut à droite de l'écran, sélectionnez le client, saisissez un titre à votre discussion, puis le corps du message.

## Réponses et pièces jointes

Dans une discussion, vous pouvez répondre en saisissant un texte en bas de page puis cliquer sur « Envoyer », mais vous pouvez également joindre une des photos présentes dans l'application pour étayer une explication, en cliquant sur le trombone.

## Notifications

Votre client est prévenu en temps réel de vos messages dans son smartphone et vous retrouverez facilement trace des discussions que vous avez eues avec lui.
